# 更新日志

本项目的所有值得注意的变更都将记录在此文件中。

格式基于 [Keep a Changelog](https://keepachangelog.com/zh-CN/1.0.0/)，
且本项目遵循 [语义化版本](https://semver.org/lang/zh-CN/)。

## [1.0.2] - 2024-12-26

### 新增
- 支持自动版本检测
- 改进发布说明生成
- 添加双语更新日志支持

### 新增
- 支持重置 Cursor IDE 设备标识

- 一键式操作界面
- 自动化多平台构建发布
