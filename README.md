# Cursor Fake

这是一个模拟Cursor IDE功能的Go语言项目。

## 功能特点

- 模拟Cursor IDE的基本功能
- 使用Go语言开发
- 简单易用的接口

## 环境要求

- Go 1.21 或更高版本
- Make

## 构建和运行

### 构建项目

```bash
make build
```

### 运行项目

```bash
make run
```

### 运行测试

```bash
make test
```

### 清理构建文件

```bash
make clean
```
